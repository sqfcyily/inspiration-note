# 💡 灵感便签 (Inspiration Note)

桌面便签/备忘录应用。基于 **Electron**、**Vanilla JS** 和 **SQLite** 构建，拥有流畅的微交互动效、自适应看板、标签/分类管理以及 Markdown 预览高亮等丰富功能。

---

## 🌟 核心功能清单

### 1. 智能便签面板
*   **双击新建**：在空白看板处双击即可在对应点击位置创建新便签。
*   **自由拖拽与缩放**：自由托拽布局下，便签支持拖动到任意位置并自由拉伸大小，位置和尺寸信息实时自动写回 SQLite 数据库。
*   **自适应画布**：看板画布大小会随着便签的拖拽和扩展自动拓宽，确保初始状态无冗余滚动条，越界时自动自适应伸缩。
*   **网格自动排列**：支持在“自由拖拽”与“网格自动排列”双模式间无缝切换，并支持按更新时间、创建时间或优先级对卡片进行排序。

### 2. 精致微交互与视觉美学
*   **毛玻璃视觉风格**：磨砂玻璃底色（`backdrop-filter: blur`）设计，包含一套雅致的多彩便签背景配色（紫色、薄荷绿、暖橙、天蓝、粉色、黄色、浅灰、白色）。
*   **多彩微动反馈**：卡片悬浮控制按钮（编辑、删除、收藏、置顶）在悬停时会执行微缩放放大（`scale(1.1)`）并配有专属主题颜色渐变反馈（删除呈高亮警示红，编辑呈主题蓝等）。
*   **文本选择冲突解决**：便签正文点击事件升级为双击（`dblclick`）打开编辑器，完美保留单点拖拽、双击选词等标准系统的文字选择与复制功能。

### 3. 便签管理与数据分析
*   **分类与表情图库**：拥有可伸缩侧边栏。支持点击表情库（Emoji Picker Popover）快速创建自定义分类，分类添加采用对称的 Grid 栅格布局防止错位。
*   **智能标签云**：保存时自动使用正则表达式抓取便签内容里的 `#话题` 字段，注册到标签库中并在卡片底部和侧边栏生成可视化的标签云分类。
*   **实时搜索高亮**：顶部提供快速检索栏，动态过滤并高亮卡片中匹配的关键词（避免特殊字符崩溃的防护性算法设计）。
*   **置顶与收藏**：支持一键星标收藏或卡片置顶，置顶卡片在任何模式下均浮在顶层。

### 4. 极致 Markdown 与安全体验
*   **Markdown + Prism.js 高亮**：卡片正文原生支持渲染 Markdown 格式文本，编辑器支持“编辑”与“实时渲染预览”双选项卡切换。集成 Prism.js 为代码块提供精美的语法高亮。
*   **无缝系统常驻**：关闭主窗口自动最小化至系统托盘，支持双击托盘恢复，右键托盘菜单支持快速呼出、新建便签及完全退出。
*   **双模主题**：支持暗色（深蓝紫调）和亮色（经典灰白）一键切换，CSS 变量动态刷新。

## 🛠️ 技术选型

*   **运行环境**：Node.js + Electron (v30.0.1)
*   **前端核心**：HTML5 / Vanilla CSS3 / JavaScript (ES6+, 零框架依赖)
*   **数据存储**：SQLite 3 (基于 `better-sqlite3` 同步本地数据库绑定)
*   **Markdown 解析**：`marked.js`
*   **代码高亮**：`prismjs`
*   **打包工具**：`electron-builder`

## 🚀 快速启动指引

### 前提条件
您的电脑上需要安装 [Node.js](https://nodejs.org/) (推荐 v18 或更高版本)。

### 1. 克隆并进入目录
```bash
cd inspiration-note
```

### 2. 安装项目依赖
由于项目包含 SQLite 原生模块编译，若提示安全限制，请使用命令行工具进行安装：
```bash
npm install
```

### 3. 调试启动应用程序
```bash
npm start
```
*提示：如果多次报错占用文件，请确认是否已有 Electron 进程在后台悬挂（可通过命令行 `taskkill /f /im electron.exe` 批量清理）。*

## 📦 应用程序打包发布

我们已经集成了 `electron-builder` 打包配置，支持 Windows 和 macOS 双平台编译：

### 1. Windows 平台打包
打包会自动为您生成**安装版 (NSIS)** 以及 **绿色免安装单文件版 (Portable)**：

#### 执行打包命令
```bash
npm run dist
```

#### 注意事项 (Windows Symlinks 特权报错解决)
由于 `winCodeSign` 压缩包中带有 macOS/Linux 系统的符号链接，Windows 普通终端在解压时会因为无权限抛出 `Cannot create symbolic link: 客户端没有特权` 错误。
*   **自动处理方案**：本工程在配置中已经做好了规避。在解压时通过 `7za.exe` 命令中附加 `-x!darwin` 剔成了不支持的平台目录，使得 Windows 编译免于特权依赖。
*   **打包成果**：编译生成的 exe 文件将输出在根目录的 `dist/` 下：
    *   **安装包**：`dist/灵感便签 Setup 1.0.0.exe` (提供安装引导、快捷方式)
    *   **绿色版**：`dist/灵感便签 1.0.0.exe` (直接双击即可流畅运行，零配置)

### 2. macOS 平台打包
为了编译生成 macOS 的 `.dmg` 安装镜像与免安装 `.app` 包，必须在一台 Mac 电脑上运行以下步骤：

#### 安装与编译
```bash
# 安装依赖 (会自编译 better-sqlite3 对应的 Mac 二进制文件)
npm install

# 运行 macOS 平台打包
npx electron-builder --mac
```

#### 打包成果
编译生成的文件将输出在根目录的 `dist/` 下：
*   **DMG 磁盘镜像**：`dist/灵感便签-1.0.0.dmg` (双击弹出标准的拖动到 Applications 的安装框)
*   **免安装 App 资源**：`dist/灵感便签.app` (可以直接运行或复制到应用程序目录)
