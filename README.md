# 💡 精灵便签 & 桌面宠物

一款基于 Electron + Vanilla JS + SQLite 构建的桌面便签与趣味桌面宠物软件。拥有流畅的微动交互、智能自适应看板、标签管理以及 Shimeji 配置驱动的桌面宠物系统。

---

## 🌟 核心功能介绍

### 📂 便签管理系统

*   **双击新建**：在看板空白区域双击即可在当前鼠标坐标处创建新便签卡片。
*   **自由拖动与缩放**：每张便签卡片均支持在看板内自由拖拽定位和拉伸缩放，大小和位置坐标实时自动保存。
*   **自适应画布**：画布尺寸随卡片拖动自适应调整，边缘越界时自动向外扩宽，保证随时拥有无限的灵感记录空间。
*   **网格自动排列**：支持在“自由拖拽模式”与“网格流式排列模式”之间一键无缝切换，并支持按更新时间、创建时间或优先级对卡片进行排序。

*   **分类与表情图库**：拥有可伸缩的侧边栏。支持点击表情库快速创建个性化分类，采用完美的对称 Grid 栅格布局呈现。
*   **智能标签云**：保存时自动使用正则表达式抓取便签内容里的 `#话题` 字段，注册到标签库中并在卡片底部和侧边栏生成可视化的标签云分类。
*   **实时搜索高亮**：顶部提供快速检索栏，动态过滤并以醒目的高亮形式展示匹配的文本关键词（包含防特殊字符崩溃的安全匹配算法）。
*   **置顶与收藏**：支持一键星标收藏或卡片置顶，置顶卡片在任何模式下均浮在顶层。

*   **Markdown + Prism.js 高亮**：卡片正文支持原生 Markdown 格式渲染，编辑器支持“编辑”与“实时渲染预览”双选项卡切换。集成 Prism.js 为代码块提供精美的语法高亮。

### 👾 桌面宠物系统

#### 1. 经典动漫角色入驻
*   内置多款精致的桌面宠物模型，包括：
    *   **路飞 (Monkey D. Luffy)**
    *   **哈利昆 (Harlequin)**
    *   **光 (Hikari)**
    *   **龙卷 (Tatsumaki)**
*   **托盘快捷菜单**：支持在 Windows 系统托盘右键菜单中一键更换当前角色，或单独显示/隐藏桌面宠物。

#### 2. Shimeji 算法级智能行为逻辑
*   **攀爬天花板与墙壁**：当宠物移动到屏幕左右两侧边缘或顶部时，能够自动贴墙直立攀爬、甚至头朝下在顶部天花板横向倒挂爬行。
*   **丰富的生活状态**：拥有深度行走、坐下发呆、呼呼大睡、倒地滑行、原地跳跃等十余种由动画配置（`animation.json`）驱动的逻辑行为，并由内置的状态机算法保证各种行为平滑过渡。
*   **边缘限制自适应**：自动实时侦测桌面分辨率边界，完美处理多显示器边界碰撞和自适应回弹。

#### 3. 互动交互与重力物理
*   **鼠标重力拖拽**：鼠标左键按下即可抓起桌面宠物并在桌面上拖拽。当被抓起悬空并释放时，宠物会自动切换到掉落动画，并在重力加速度物理计算下坠落回桌面或吸附到就近的墙壁上。
*   **穿透模式**：闲置或运动时自动开启鼠标穿透（`setIgnoreMouseEvents`），绝不干扰您的日常电脑办公与操作。

#### 4. 趣味对话与气泡台词
*   **自言自语**：在宠物处于闲置（Idle）或坐下状态时，每隔一定时间会随机在头顶弹出一个半透明对话框进行“自言自语”。
*   **个性化台词库**：为每个角色（路飞、哈利昆、光、龙卷）量身定制了多达十几条与其性格完全吻合的专属中文台词，随时为您提供陪伴与乐趣。
